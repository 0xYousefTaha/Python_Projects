# CLI Calculator

A simple command-line calculator written in Python. This calculator supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features

- Addition, subtraction, multiplication, and division
- Input validation for numbers and operations
- Handles division by zero errors gracefully
- Option to continue calculations with the result or start a new calculation
- ASCII art calculator logo

## Usage

1. Run the calculator from the command line:

    ```sh
    python Calculator.py
    ```

2. Follow the prompts to enter numbers and select operations.

3. After each calculation, choose to continue with the result or start a new calculation.
